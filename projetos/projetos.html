First Lego League

Nesse projeto fizemos um robo para a FLL( First Lego League 2021/2022) onde ele tinha como objetivo completar objetivos da mesa, representando a TainhaTec da Escola S. Foi totalmente feito em lego e foi programado usando Scratch ( Uma das linguagens usadas no spike que foi o bloco de comando que usamos )

Bone para deficientes visuais

Esse projeto foi feito na startup Idocode com o foco de ajudar deficientes visuais com um boné onde tinha um sonar em sua ponta para detectar quando ele estivesse perto da parede. Para realizar esse projeto foi usado raspberrypi, sonar e uma powerbank para manter ligado o boné. Além disso foi usado python, ele está postado no Github

Leds remotamente com esp8266

Esse abajur foi refeito usando esp8266 nodeMCU com C++, um projeto facilmente adaptavel onde o usuario usando o app Blynk pode desligar e ligar os leds ligados no esp, codigo postado no Github
