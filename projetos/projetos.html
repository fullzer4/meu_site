Bone deficientes visuais

Foi feito na startup idocode, e esse projeto tem o principio de ajudar deficientes visuais atraves de um boné com sonar onde apita quando chega a certa distancia.

Para a confecção desse bone foi usado raspeberripy, sonar, powerbank e alguns fios e usamos a linguagem python para verificar a distancia onde esta no meu Github

Controladora leds Wifi esp NodeMCU

Nesse projeto foi feito com que apenas voce tendo um celular voce consiga contralar leds a distancia com extrema facilidade.

Foi usado esp8266 NodeMCu, uma luminaria e pilhas (toda a programação foi feita em C++ junto com a biblioteca do app blynk), o app usado para desligar e ligar foi o blynk onde tu sicronizava o esp com seu token no codigo e ele comunicava com o app assim possibilitando esse uso.
